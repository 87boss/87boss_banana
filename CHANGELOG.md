### 問題修復
- **(2026-01-23)** 修復終端無限循環錯誤
  - 啟動 Backend 服務 (port 8766) 解決 Vite proxy `ECONNREFUSED` 錯誤
  - 終端不再顯示數百次 proxy 錯誤訊息
- **(2026-01-23)** 修復桌面縮圖不斷閃爍問題
  - 在解碼回調中添加重複檢查機制
  - 避免同一解碼圖片被重複添加到桌面
- **(2026-01-23)** 修復解碼縮圖位置覆蓋原圖問題
  - 修改為在 `setDesktopItems` 回調內部計算空閒位置
  - 使用最新桌面狀態而非閉包中的舊狀態
  - 解決 React 閉包陷阱導致的位置計算錯誤

### 新增
- 完善開源專案規範管理
  - 最佳化 .gitignore 規則，新增可執行檔案、臨時檔案、備份檔案等忽略規則
  - 為空目錄新增 .gitkeep 檔案，確保目錄結構完整
  - 統一前後端版本號為 0.2.4
  - 建立 CHANGELOG.md 和 CONTRIBUTING.md 檔案

## 版本號說明

- **主版本號 (MAJOR)**：不相容的 API 修改或重大架構變更
- **次版本號 (MINOR)**：向下相容的功能性新增
- **修訂號 (PATCH)**：向下相容的問題修正

## 變更型別

- **新增**：新功能
- **改進**：對現有功能的改進
- **棄用**：即將移除的功能
- **移除**：已移除的功能
- **問題修復**：Bug 修復
- **安全**：安全相關的修復
