## 🎨 畫布功能介紹

企鵝工坊的**畫布（Pebbling Canvas）**是一個視覺化節點編輯器，讓 AI 影象創作流程變得直觀且強大。

### 核心特性

**📦 豐富的節點型別**

- **創意節點（Creative）** - 從創意庫直接拖入畫布使用
- **圖片節點（Image）** - 上傳或載入圖片作為輸入源
- **提示詞節點（Prompt）** - 編寫和組合提示詞
- **文位元組點（Text）** - 新增文字說明或標註
- **儲存節點（SaveImage）** - 儲存生成的圖片到桌面
- **多視角節點（MultiAngle）** - 3D 視角控制，支援旋轉、俯仰、縮放
- **編輯節點（Edit）** - 圖生圖，基於輸入圖片生成新圖
- **去背景節點（RemoveBG）** - AI 智慧摳圖
- **放大節點（Upscale）** - 圖片超解析度放大
- **尺寸調整節點（Resize）** - 調整圖片尺寸和比例
- **LLM 節點** - 文字生成和處理，支援自定義 System Prompt

**🔗 視覺化連線流程**

- 節點之間透過連線建立資料流
- 支援多輸入：一個節點可以接收多個上游輸入
- 自動傳遞：提示詞、圖片、引數自動在節點間傳遞
- 批次生成：支援設定生成數量，一鍵批次執行

**💾 畫布管理**

- **多畫布支援** - 建立多個畫布，獨立管理不同專案
- **自動儲存** - 畫布狀態實時自動儲存
- **匯入匯出** - 畫布可匯出為 JSON 檔案分享
- **預設系統** - 將常用節點組合儲存為預設，快速複用

**🎯 工作流示例**

```
創意節點 → 多視角節點 → 編輯節點 → 儲存節點
     ↓
提示詞節點 → 編輯節點 → 放大節點 → 儲存節點
```

**⚙️ 高階功能**

- **多視角控制**：透過滑塊精確控制 3D 視角
  - 水平旋轉：0° - 360°
  - 垂直俯仰：-90° - 90°
  - 距離縮放：0.5x - 3.0x
  - 支援簡單/詳細兩種提示詞模式

- **圖片後設資料顯示**：自動提取並顯示
  - 圖片尺寸（寬 x 高）
  - 檔案大小
  - 圖片格式

- **執行控制**
  - 單節點執行：只執行當前節點
  - 批次執行：設定數量批次生成
  - 停止執行：隨時中斷執行
  - 進度顯示：實時顯示執行狀態

**🎪 與桌面聯動**

- 畫布生成的圖片自動同步到桌面
- 創意庫內容可直接拖入畫布使用
- 畫布可建立對應的桌面資料夾

## 🐛 Bug 修復

### Windows 打包最佳化

**問題**：打包後的應用工作列顯示 Electron 預設圖示，而非企鵝 LOGO

**解決方案**：
1. 生成標準 ICO 格式圖示（支援 16x16、32x32、48x48、256x256）
2. 新增 `afterPack` 構建鉤子
3. 使用 `rcedit` 工具將圖示嵌入到 .exe 檔案
4. 在生產環境隱藏選單欄，提供更沉浸的使用體驗

**改進效果**：
- ✓ 工作列正確顯示企鵝圖示
- ✓ 安裝程式圖示統一
- ✓ 介面更加簡潔專業

## 📝 使用建議

### 首次使用畫布

1. 點選主介面右上角的「畫布」按鈕
2. 建立新畫布或開啟已有畫布
3. 從左側邊欄拖入節點到畫布
4. 連線節點建立工作流
5. 點選節點上的「執行」按鈕開始生成

### 典型應用場景

**場景一：批次多視角生成**
```
創意節點 → 多視角節點（設定不同角度）→ 儲存節點
```

**場景二：圖片精修流程**
```
圖片節點 → 去背景 → 放大 → 尺寸調整 → 儲存節點
```

**場景三：創意擴充套件**
```
提示詞節點 → 編輯節點（多個，不同引數）→ 儲存節點
```
