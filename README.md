## ✨ 核心特性

### 🖥️ 桌面級管理體驗
像整理桌面一樣管理你的 AI 作品。拖拽、分組、堆疊，一切盡在掌控。

### 📁 智能文件夾 & 堆疊
- **文件夾** — 按項目歸類，清晰有序
- **智能堆疊** — 同系列作品自動聚合，節省空間

### 🖱️ 自然拖放交互
- 從電腦直接拖拽圖片/文件夾到工作台
- 拖拽調整位置，所見即所得
- 多選批量操作，效率倍增

### 📚 創意庫系統
內置多種創意模板，一鍵應用專業提示詞，小白也能出大片。

### 🔄 完整創作閉環
生成 → 預覽 → 再編輯 → 重新生成 → 管理保存，全流程無縫銜接。

## 🚀 快速開始

### 方式一：一鍵啟動（推薦）

```bash
# 1. 首次使用，雙擊運行
install.bat

# 2. 以後每次啟動，雙擊運行
start.bat

# 3. 自動打開瀏覽器
http://127.0.0.1:8765
```
每次更新後都要如上操作一遍！！！

### 方式二：手動啟動

```bash
# 1. 安裝前端依賴
npm install

# 2. 安裝後端依賴
cd backend-nodejs
npm install
cd ..

# 3. 構建前端
npm run build

# 4. 啟動 Node.js 後端服務
cd backend-nodejs
node src/server.js

# 5. 打開瀏覽器
http://127.0.0.1:8765
```

### 環境要求

- Node.js 18 或更高版本
- Windows 10/11（其他系統需手動啟動）
